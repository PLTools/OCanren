Relational API

type 'a groundi = ( 'a, 'a groundi ) t ilogic

A synonym for injected list

type 'a injected = 'a groundi

Conversions between data types

val of_list : ( 'a -> 'b ) -> 'a GT.list -> 'b ground

The of_list l converts regular OCaml list l into isomorphic OCanren ground list. See to_list for a reverse conversion.

val to_list : ( 'a -> 'b ) -> 'a ground -> 'b GT.list

The to_list g converts OCanren list g into regular OCaml list. See of_list for a reverse conversion.

val inj : ( 'a -> 'b ) -> 'a ground -> 'b logic

The inj x makes a logic list from a ground one. See logic_to_ground_exn for a partial reverse conversion.

val logic_to_ground_exn : ( 'a -> 'b ) -> 'a logic -> 'b ground

Converts a logic list to ground one.

  • raises [Failure]

    when a logic variable occurs inside a list.

val list : 'a GT.list -> 'a groundi

Make injected list from ground one of injected elements. The reverse conversion is availble only through reifiers (see Reifiers for details).

Constructors

val nil : unit -> 'a groundi

A logical empty list. Extra unit parameter prevents weak type variables.

val cons : 'a -> 'a groundi -> 'a groundi

A dual for cons (a.k.a. ::) constructor.

val (%) : 'a -> 'a groundi -> 'a groundi

Infix synonym for cons

val (%<) : 'a -> 'a -> 'a groundi

x %< y is a synonym for cons x (cons y (nil ()))

val (!<) : 'a -> 'a groundi

!< x is a synonym for cons x (nil ())
