Module OCanren.Reifier

type ('a, 'b) t = ( 'a -> 'b ) Env.Monad.t

Reifier from type `'a` into type `'b` is an `'a -> 'b` function dipped into the Env.t monad, will see how it plays later.

Perhaps, it is possible to not expose the reifier type and make it itself a monad or something else that composes nicely, but we haven't figured out yet.

A few predefined reifiers from which other reifiers will be composed

val reify : ( 'a ilogic, 'a logic ) t
val prj_exn : ( 'a ilogic, 'a ) t

This one projects implicit logic into the underlying type, raising exception Not_a_value if it finds a variable

val prj : ( int -> 'a ) -> ( 'a ilogic, 'a ) t
val compose : ( 'a, 'b ) t -> ( 'b, 'c ) t -> ( 'a, 'c ) t

Composition of two reifiers

Reifier is a profunctor. Below are two combinators to compose reifiers with regular functions
